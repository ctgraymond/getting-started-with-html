Important Concepts

What is a webpage?
A webpage is a text document that is written in HyperText Markup Language[HTML] Web browsers read these text documents, interpret them, and display the results on screen.

Computers are stupid.
Computers will execute whatever commands you give them, regardless of the outcome. It is important that programmers communicate in a way that computers understand, and double-check tags to ensure that they are complete/correct.

Basic HTML Vocabulary/Tags
-Bold Text: The inline tag <b></b> can be used to bold text.
-Italicized Text: The inline tag <em></em> can be used to italicize text.
-Insert Images: The inline void tag <img src="source"> can be used to insert an image into the webpage.
-Hyperlink: The inline anchor tag <a href="source"> </a> can be used to create a hyperlink within text.
-Span/Div: <span></span> and <div></div> will be important in allocating space later on; <span> is an inline tag and <div> is a block tag.
